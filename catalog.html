Каталог решений

Готовые модули для автоматизации учета

📊 Модуль "Кадры"

Полное ведение кадрового учета: личные карточки сотрудников, трудовые договоры, приказы, штатное расписание.

Цена: Входит в базовый пакет

⏱️ Модуль "Табель"

Автоматизированный учет рабочего времени с возможностью учета сменности, отпусков, больничных и командировок.

Цена: Входит в базовый пакет

💰 Модуль "Расчет зарплаты"

Расчет окладов, премий, надбавок. Поддержка прогрессивной шкалы НДФЛ. Автоматический расчет страховых взносов.

Цена: Входит в базовый пакет

📄 Модуль "Отчетность"

Формирование всех необходимых документов: расчетные листки, платежные ведомости, справки 2-НДФЛ, формы для ПФР и ФСС.

Цена: Входит в профессиональный пакет

🏦 Модуль "Банк-клиент"

Интеграция с банковскими системами для автоматической выгрузки платежных поручений на перечисление зарплаты.

Цена: +5 000 ₽/мес

📊 Модуль "Аналитика"

Расширенная аналитика по фонду оплаты труда: отчеты по подразделениям, динамика изменений, прогнозирование.

Цена: +3 000 ₽/мес
